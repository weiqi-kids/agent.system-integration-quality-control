系統整合品質管制 — 自動化安全掃描與 SEO 檢測流程

檢測項目說明

SSDLC 安全開發生命週期

檢查專案是否具備安全開發基礎：CI/CD 設定、依賴管理、密鑰掃描、分支保護等。

SAST 靜態源碼掃描

使用 Semgrep 對原始碼進行靜態分析，偵測注入攻擊、XSS、不安全的加密等安全漏洞。

弱點掃描

使用 Trivy 掃描依賴套件的已知漏洞 (CVE)，依嚴重度分級：Critical / High / Medium。

滲透測試

使用 OWASP ZAP 與 Nuclei 對上線網站進行動態安全測試，模擬攻擊者行為。

連結檢查

掃描網站所有內部與外部連結，偵測 404 斷連、重導向異常等問題。

SEO / AEO 檢測

Meta 標籤、Schema 結構化資料、SGE/AEO 標記、E-E-A-T 信號、Core Web Vitals、YMYL 合規。
